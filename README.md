# Elotus movies

It is a movies app using the [The Movie Database API](https://developers.themoviedb.org/3).

## Development

- Install dependencies

```bash
yarn install 
```

- Run local development

```bash
yarn dev 
```

- Build project

```bash
yarn build 
```

## Stacks

- NextJS
- React-Query
- TailwindCSS
- Axios
